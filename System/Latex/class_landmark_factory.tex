\doxysubsection{Detailed Description}
Abstract factory class for creating \doxylink{class_landmark}{Landmark} structures. 

{\ttfamily \doxylink{class_landmark_factory}{Landmark\+Factory}} serves as a base class for specific factories that create landmark types like parks, colosseums, and pantheons.
