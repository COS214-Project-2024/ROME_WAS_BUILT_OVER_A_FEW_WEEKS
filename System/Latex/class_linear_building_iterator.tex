\doxysubsection{Detailed Description}
A concrete iterator that provides linear traversal over a collection of {\ttfamily \doxylink{class_residential_complex}{Residential\+Complex}} buildings. 

The {\ttfamily \doxylink{class_linear_building_iterator}{Linear\+Building\+Iterator}} iterates over a vector of {\ttfamily \doxylink{class_residential_complex}{Residential\+Complex}} pointers in a linear fashion, starting from the beginning of the collection and moving sequentially to the end.
