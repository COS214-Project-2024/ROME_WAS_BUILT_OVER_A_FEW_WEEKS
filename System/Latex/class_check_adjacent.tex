\doxysubsection{Detailed Description}
Utility class to check adjacent structures in a 2D map of \doxylink{class_city_structure}{City\+Structure} pointers. 

The \doxylink{class_check_adjacent}{Check\+Adjacent} class provides functionality to count and perform actions on non-\/null adjacent structures around a specified position in the map.

Checks adjacent cells around a given position in the map for non-\/null structures. 

This method iterates over the cells adjacent to the specified coordinates {\ttfamily (x, y)} in the {\ttfamily map}. If an adjacent cell contains a non-\/null structure, the method increments the count and performs an action on that structure.
