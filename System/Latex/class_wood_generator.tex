\doxysubsection{Detailed Description}
A decorator for generating wood resources in the city. 

The \doxylink{class_wood_generator}{Wood\+Generator} class extends the \doxylink{class_generator_decorator}{Generator\+Decorator} and provides functionality to generate wood resources and manage the wood factory within the city map.
