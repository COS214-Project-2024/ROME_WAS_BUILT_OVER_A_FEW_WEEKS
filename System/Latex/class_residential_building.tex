\doxysubsection{Detailed Description}
Represents a base class for different types of residential buildings. 

The {\ttfamily \doxylink{class_residential_building}{Residential\+Building}} class defines common functionality and properties for residential buildings, including capacity, cost, and the ability to place and remove structures.
