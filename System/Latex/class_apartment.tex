\doxysubsection{Detailed Description}
Represents a type of residential building\+: \doxylink{class_apartment}{Apartment}. 

The {\ttfamily \doxylink{class_apartment}{Apartment}} class inherits from {\ttfamily \doxylink{class_residential_building}{Residential\+Building}} and defines an apartment-\/specific configuration with preset costs for construction resources.
