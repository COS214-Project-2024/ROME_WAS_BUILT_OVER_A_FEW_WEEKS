\doxysubsection{Detailed Description}
Represents an office building with specific costs and radius. 

The \doxylink{class_office}{Office} class inherits from \doxylink{class_commercial_building}{Commercial\+Building} and provides specific values for cost and required materials (wood, steel, and concrete).
